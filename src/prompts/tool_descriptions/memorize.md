## Description
Memorizes a piece of information.

## When to use
Always use it when, in the process of executing the user request, you find a fact, mistake, solution, or idea you could use in the future to solve a problem or to complete a task.

## Arguments

- `information` (string, required): The piece of information to memorize.
- `information_type` (string, required): Category of the information. Must be one of: `fact`, `idea`, `mistake`, `solution`.

## Returns

A confirmation message indicating the information was saved, or an error message if the operation failed.
